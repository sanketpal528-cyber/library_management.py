# Changelog

All notable changes to LibraryMS are documented here.

---

## [1.2.0] - 2026-07-03

### Added
- 3 new pages: Catalogue (visual book cards), About (orbit animation + timeline), Settings
- Futuristic animated background: particles, circuit lines, neon waves, mouse interaction
- Loading screen with animated progress bar
- Typing animation in hero section
- Scroll fade-in for cards
- Pulsing neon borders
- Export / Import data as JSON backup
- Accent color picker in Settings
- Compact mode toggle

### Fixed
- Data not persisting after page refresh (syncData() fix)
- showSection override causing infinite loop on new pages

---

## [1.1.0] - 2026-05-13

### Added
- Dark theme with glassmorphism UI
- Poppins + Inter Google Fonts
- Font Awesome 6 icons throughout
- Animated stat cards with glow effects
- GitHub Actions CI/CD deploy workflow

### Changed
- Redesigned navbar with gradient brand text
- Improved table row hover with neon left border
- Enhanced button glow effects

---

## [1.0.0] - 2026-05-12

### Added
- Book Management: Add, Edit, Delete, Search books
- Member Management: Register, Edit, Delete, Search members
- Borrow / Return with 14-day due dates
- Fine calculation (₹5/day overdue)
- Live fine preview on Record ID input
- Records with filter (All / Active / Returned / Overdue)
- Dashboard with 6 live stat cards
- Reports: top borrowed books + overdue list
- localStorage persistence

---

## [0.1.0] - 2026-05-11

### Added
- Initial Python console-based version
- Basic book add/view functionality
